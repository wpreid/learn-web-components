Long Method

Like a very larger burger, a long method is too much to digest.

How long must a method be to exhibit the Long Method smell? If one cannot quickly and easily understand what a method does and how the method does it, it is a long method.

Shorter methods provide these benefits:

  • Readable Code - Shorter methods are easier to read and comprehend.
  • Shared Code - Shorter methods provide greater opportunity for fine-grained code sharing.
  • Easier Testing - Shorter method tend to be easier to microtest (AKA unit test).
  • Safer Editing - Shorter methods have fewer variables and states to manage.

We suggest that people not make methods long, dense, wide, or in any other way inconvenient for their friends to work with. Everyone is in a hurry so don't force them to be ultra-careful.
